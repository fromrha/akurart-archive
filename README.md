# Akurart Archive

**Akurart Archive** adalah sebuah *running record* dari serpihan, fragmen, dan momen-momen yang ditemukan dalam musik, film, dan budaya visual. Mulai dari shoegaze yang pudar hingga seluloid yang terpapar matahari, kami mendokumentasikan nada-nada yang tidak selalu mendapatkan ruang tayang yang bersih.

Web ini dibangun sebagai ruang alternatif bagi mereka yang mencari makna lebih dalam dari sekadar informasi cepat.

### Value Proposition
Keunikan Akurart Cinema terletak pada pendekatan **naratif dan personal** dalam membahas musik, film, dan budaya. Kami tidak berfokus pada tren atau rating, melainkan pada cerita, konteks, dan pengalaman manusia di balik karya. Akurart Cinema menawarkan konten yang tenang, reflektif, dan berjarak dari clickbait, menjadikannya ruang alternatif bagi pembaca yang ingin memahami budaya secara lebih dalam dan jujur.

### Tech Stack Details
- **Core Framework**: [Next.js 16](https://nextjs.org/) (App Router) - Framework React paling modern untuk production.
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Typed superset dari JavaScript untuk kode yang lebih aman.
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS framework untuk styling cepat dan responsif.
- **Content Management System (CMS)**: [Sanity.io](https://www.sanity.io/) - Platform konten terstruktur untuk mengelola artikel dan media secara real-time.
- **Animations**: [GSAP](https://gsap.com/) - Library animasi standar industri untuk interaksi web yang halus dan *ScrollReveal*.

## Instalasi & Menjalankan Lokal

Ikuti langkah berikut untuk menjalankan project di local machine kamu:

1. **Clone repository**
   ```bash
   git clone https://github.com/username/akurart-archive.git
   cd akurart-archive
   ```

2. **Install dependencies**
   ```bash
   npm install
   # atau
   yarn install
   # atau
   pnpm install
   ```

3. **Setup Environment Variables**
   Pastikan kamu memiliki akses ke Sanity project ID dan dataset. Buat file `.env.local` dan isi konfigurasi yang diperlukan:
   ```env
   NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
   NEXT_PUBLIC_SANITY_DATASET=your_dataset
   ```

4. **Jalankan Development Server**
   ```bash
   npm run dev
   ```

5. **Buka Browser**
   Buka [http://localhost:3000](http://localhost:3000) untuk melihat hasilnya.
